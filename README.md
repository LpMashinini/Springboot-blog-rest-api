# Login and Registration REST API

A secure **Login and Registration REST API** built with **Spring Boot, Spring Security, Spring JPA, Hibernate, and MySQL**, fully containerized with **Docker** and integrated with **CI/CD pipelines** for automated builds and deployments.

---

## 🚀 Features
- User registration with validation
- Secure login 
- Role-based access control 
- Password encryption using BCrypt
- RESTful API endpoints
- MySQL database integration with Hibernate & JPA
- Configurable Spring Security setup
- **Dockerized application** for easy deployment
- **CI/CD pipeline** for automated testing and deployment

---

## 🛠️ Tech Stack
- **Spring Boot** – Application framework
- **Spring Security** – Authentication & authorization
- **Spring JPA / Hibernate** – ORM for database operations
- **MySQL** – Relational database
- **Docker** – Containerization
- **CI/CD** – Automated build & deployment pipeline
- **Maven** – Build tool
